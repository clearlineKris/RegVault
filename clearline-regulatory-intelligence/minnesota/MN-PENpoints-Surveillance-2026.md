## Minnesota Security Framework

### Minn. Stat. § 342 (OCMM Rules) — Premises Security
- Camera coverage: all entry/exit, all storage, all sales, all cultivation, all processing, all loading/unloading
- **90-day minimum footage retention** (Minnesota matches Ohio, Michigan, Missouri, Pennsylvania, and New York at the 90-day standard)
- Visitor log: name, date, time, purpose, escort
- Security plan on file with OCMM; updated when facility changes
- 24/7 central alarm monitoring

---

## Minnesota's Starting Point Advantage

Unlike Michigan (2019 installs), Missouri (2018 medical installs), or Oklahoma (2020 installs), Minnesota's adult-use program launched in 2023. Operators who installed security systems at or after launch have newer hardware with appropriate retention capacity — IF they specified 90-day retention at installation.

**The risk for new Minnesota operators:** Security vendors may not default to 90-day configuration. A system installed in 2023 is still non-compliant if it was configured for 30-day retention. Newness of hardware doesn't equal compliance — the configuration matters.

## Compliance Hardening Checklist — Minnesota Security

- [ ] DVR/NVR specified and confirmed for 90-day retention at current camera count + resolution
- [ ] Footage retrieval from Day 85 tested and confirmed
- [ ] Camera coverage audit current against full facility footprint
- [ ] Security plan filed with OCMM; updated for any facility modifications
- [ ] Alarm monitoring contract current and active monitoring confirmed
- [ ] Panic buttons at all POS + cash handling
- [ ] Visitor log — digital preferred, all required fields
- [ ] Staff training on restricted access and visitor escort documented
