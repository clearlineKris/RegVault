## Ohio Regulatory Framework Overview

**Governing Authority:** Ohio Division of Cannabis Control (ODCC), under ORC Chapter 3780 (adult-use, effective 2024) and ORC Chapter 3796 (medical marijuana). Adult-use retail sales launched June 7, 2024.

Ohio operates a dual-track system: established medical program (Ohio Medical Marijuana Control Program / OMMCP) and the newer adult-use framework under ODCC. Many license holders operate across both tracks.

## Section 1: Licensing Obligations

### 1.1 Statutory Foundation
- **ORC § 3780.10**: Adult-use cultivator license categories (Type 1, Type 2)
- **ORC § 3780.11**: Adult-use processor license
- **ORC § 3780.12**: Adult-use retail dispensary license (Level I–V)
- **ORC § 3780.15**: Testing laboratory license
- **ORC § 3796.09**: Medical marijuana cultivator license (Class I, II)
- **ORC § 3796.10**: Medical marijuana processor license
- **ORC § 3796.11**: Medical marijuana dispensary license
- **Ohio Admin. Code 3780-2-01** et seq.: Adult-use license application and operational requirements

### 1.2 License Categories — Adult-Use (2026 Operative)

| License Type | ORC Ref | Key Limit / Condition |
|---|---|---|
| Adult-Use Cultivator Type 1 | § 3780.10(A)(1) | ≤ 3,000 sq ft canopy (indoor); ≤ 2 acres (outdoor) |
| Adult-Use Cultivator Type 2 | § 3780.10(A)(2) | > 3,000 sq ft canopy; scaled fees apply |
| Adult-Use Processor | § 3780.11 | Extraction method disclosure to ODCC required |
| Adult-Use Dispensary Level I | § 3780.12 | Lower-volume; community proximity restrictions |
| Adult-Use Dispensary Level II–V | § 3780.12 | Volume-scaled; Level V = highest volume authorization |
| Adult-Use Testing Laboratory | § 3780.15 | ISO 17025 accreditation; independence requirements |
| Adult-Use Transporter | § 3780.13 | Required for all inter-facility product movement |

### 1.3 Home Cultivation — ORC § 3780.34
- Adults 21+ may grow up to **6 plants** (3 mature, 3 immature) at primary residence
- Home cultivation does not require a state license
- Home grow product may not be sold, transferred for value, or commercially processed

### 1.4 Compliance Checklist — Licensing

- [ ] Current ODCC license certificate posted at licensed premises (Admin. Code 3780-2-05(A))
- [ ] Ownership and beneficial interest disclosure current (ORC § 3780.10(C))
- [ ] Background check current for all principal officers and owners (Admin. Code 3780-2-03)
- [ ] Renewal application filed within annual renewal window (Admin. Code 3780-2-07)
- [ ] Local authorization/zoning approval maintained alongside state license (ORC § 3780.33)
- [ ] Dual-track operators: medical OMMCP license maintained separately where applicable

## Section 2: Seed-to-Sale Tracking

### 2.1 Statutory Foundation
- **ORC § 3780.18**: Mandatory seed-to-sale tracking system participation
- **Ohio Admin. Code 3780-4-02**: Metrc tracking requirements for adult-use
- **Ohio Admin. Code 3796-5-01**: Metrc requirements for medical marijuana (parallel obligations)

### 2.2 Metrc Obligations — 2026

| Obligation | Citation | Deadline |
|---|---|---|
| Plant tag at propagation | Admin. Code 3780-4-02(A) | Day of propagation |
| Harvest batch creation | Admin. Code 3780-4-02(B) | Day of harvest |
| Transfer manifest (pre-departure) | Admin. Code 3780-4-03 | Before vehicle departure |
| Inventory reconciliation | Admin. Code 3780-4-02(D) | Weekly; documented |
| Dual-track Metrc reconciliation | Admin. Code 3780-4-02(F) | Daily cross-check for dual-license operators |
| Discrepancy report | Admin. Code 3780-4-02(E) | Within 3 business days of discovery |

### 2.3 Ohio-Specific: Dual-Track Metrc Complexity

Ohio's dual-track (medical + adult-use) creates a Metrc complexity not present in single-track states. Dual-license operators run parallel Metrc environments — adult-use inventory and medical inventory must remain segregated in both physical and digital tracking.

### 2.4 Compliance Checklist — Tracking

- [ ] All plants tagged at first propagation under ORC § 3780 rules (Admin. Code 3780-4-02(A))
- [ ] Harvest batches created same-day (Admin. Code 3780-4-02(B))
- [ ] Transfer manifests generated before transport (Admin. Code 3780-4-03)
- [ ] Weekly physical count reconciled against Metrc (Admin. Code 3780-4-02(D))
- [ ] Medical and adult-use inventory physically and digitally segregated (Admin. Code 3780-4-02(F))
- [ ] Discrepancy reports filed within 3 business days (Admin. Code 3780-4-02(E))

## Section 3: Security & Access Control

### 3.1 Statutory Foundation
- **Ohio Admin. Code 3780-3-06**: Security requirements for adult-use facilities
  - Camera coverage: all entry/exit, all storage, all sales areas
  - Retention: **90-day** minimum (higher than most states)
  - Visitor log and escort requirements
- **Ohio Admin. Code 3780-3-07**: Alarm systems — 24/7 monitoring, panic buttons, motion detection

### 3.2 Security Requirements Matrix

| Area | Camera Required | Retention | Restricted Access |
|---|---|---|---|
| All exterior entry/exit | ✅ | 90 days | ✅ Badged only |
| Retail sales floor | ✅ | 90 days | ✅ Employees + licensed visitors |
| Vault/storage areas | ✅ | 90 days | ✅ Managers only |
| Processing/extraction rooms | ✅ | 90 days | ✅ Licensed processors only |
| Cultivation canopy | ✅ | 90 days | ✅ Badged only |
| Loading/unloading | ✅ | 90 days | ✅ Manifest-verified |

### 3.3 Compliance Checklist — Security

- [ ] DVR/NVR system retaining footage for minimum **90 days** (Admin. Code 3780-3-06(B)) — Ohio's requirement exceeds most states
- [ ] Storage capacity verified to sustain 90-day retention without overwrite
- [ ] Camera coverage audit current with facility footprint (all modifications reflected)
- [ ] 24/7 alarm monitoring contract current (Admin. Code 3780-3-07(B))
- [ ] Visitor log — date, time, name, purpose, escort employee — maintained (Admin. Code 3780-3-06(C))
- [ ] Annual security plan filed with ODCC (Admin. Code 3780-3-06(D))

## Section 4: Pesticide Compliance

### 4.1 Statutory Foundation
- **Ohio Admin. Code 3780-3-12**: Chemical input and pesticide requirements
  - Only EPA-registered pesticides permitted
  - Ohio Department of Agriculture (ODA) pesticide applicator license required for certain applications
  - Application log required: product, EPA reg. no., date, applicator, rate, target pest
  - 48-hour re-entry interval in canopy areas post-application
- **ODA Pesticide Regulation Program**: Coordinates with ODCC on cannabis-specific pesticide enforcement

### 4.2 Key Ohio-Specific Pesticide Condition: ODA License Requirement
Ohio requires that commercial pesticide applications (non-general-use) be made by an ODA-licensed pesticide applicator. This creates a compliance layer not present in all states — verify your IPM contractor and in-house applicators hold current ODA licenses.

### 4.3 Compliance Checklist — Pesticides

- [ ] All pesticides verified as EPA-registered and permitted for cannabis or food-crop use (Admin. Code 3780-3-12(A))
- [ ] Application log current and specific (product + EPA reg. no. + date + applicator + rate + target pest + zone)
- [ ] Commercial applications performed by ODA-licensed pesticide applicator (Ohio Pesticide Law, ORC § 921.02)
- [ ] 48-hour re-entry intervals documented and access restricted (Admin. Code 3780-3-12(C))
- [ ] IPM plan on file with ODCC (Admin. Code 3780-3-12(D))
- [ ] Pre-harvest pesticide residue testing included in testing protocol

## Section 5: Packaging & Labeling

### 5.1 Statutory Foundation
- **ORC § 3780.20**: Packaging and labeling requirements for adult-use marijuana products
- **Ohio Admin. Code 3780-3-10**: Product packaging standards
  - Child-resistant packaging (16 CFR § 1700.15)
  - Universal symbol required
  - THC content per serving and per package
  - "For use only by adults 21 and over" statement
  - Licensed manufacturer name + ODCC license number
  - Batch/lot number linked to Metrc

### 5.2 Required Label Elements — 2026

| Element | Requirement | Citation |
|---|---|---|
| Ohio universal THC symbol | Required | Admin. Code 3780-3-10(A)(1) |
| "Contains Marijuana — For Adults 21+" | Required | Admin. Code 3780-3-10(A)(2) |
| Net weight or volume | Required | Admin. Code 3780-3-10(A)(3) |
| Total THC per serving (mg) | Required — edibles | Admin. Code 3780-3-10(A)(4) |
| Total THC per package (mg) | Required | Admin. Code 3780-3-10(A)(5) |
| Licensed manufacturer + ODCC license no. | Required | Admin. Code 3780-3-10(A)(6) |
| Batch/lot number (Metrc-linked) | Required | Admin. Code 3780-3-10(A)(7) |
| Child-resistant packaging certification | Required | Admin. Code 3780-3-10(B) |
| "Keep out of reach of children and pets" | Required | Admin. Code 3780-3-10(A)(8) |

## Section 6: Testing Requirements

### 6.1 Statutory Foundation
- **ORC § 3780.19**: Mandatory third-party laboratory testing
- **Ohio Admin. Code 3780-5**: Testing laboratory requirements
  - ISO 17025 accreditation required
  - Independence requirements — no financial relationship with clients
- **Ohio Admin. Code 3780-5-02**: Required test panels for adult-use products

### 6.2 Required Testing Matrix

| Test Category | Required For | Action Level | Citation |
|---|---|---|---|
| Potency (cannabinoid profile) | All products | Label accuracy ±15% | Admin. Code 3780-5-02(A) |
| Microbials (aerobic count, e-coli, salmonella) | All products | ODCC action limits | Admin. Code 3780-5-02(B) |
| Mycotoxins | Flower, concentrates | 20 ppb total | Admin. Code 3780-5-02(C) |
| Residual solvents | Concentrates, vapes | ODCC limits per solvent | Admin. Code 3780-5-02(D) |
| Pesticide residues | All products | ODA/ODCC action limits | Admin. Code 3780-5-02(E) |
| Heavy metals | All products | USP <232> limits | Admin. Code 3780-5-02(F) |
| Water activity / moisture | Flower | ≤ 0.65 Aw | Admin. Code 3780-5-02(G) |

## Section 7: Transportation

### 7.1 Statutory Foundation
- **Ohio Admin. Code 3780-4-03**: Transportation requirements
  - Manifest required before departure
  - Enclosed, locked cargo compartment
  - ODCC-badged driver required
  - Manifests must match Metrc transfer entries
- **ORC § 3780.13**: Transporter license — separate license required for all inter-facility movement

### 7.2 Ohio-Specific: Separate Transporter License
Unlike some states where the originating licensee can self-transport, Ohio requires a separate ODCC Transporter license for all inter-facility movement. This applies to all adult-use transfers.

### 7.3 Compliance Checklist — Transport

- [ ] ODCC Transporter license current and on file for all inter-facility moves (ORC § 3780.13)
- [ ] Metrc transfer created before departure (Admin. Code 3780-4-03(A))
- [ ] Enclosed, locked cargo compartment verified (Admin. Code 3780-4-03(B))
- [ ] Driver ODCC badge current (Admin. Code 3780-4-03(C))
- [ ] Paper manifest matches Metrc — no discrepancies (Admin. Code 3780-4-03(D))

## Section 8: Waste Handling

### 8.1 Statutory Foundation
- **Ohio Admin. Code 3780-3-15**: Marijuana waste disposal
  - Render "unusable and unrecognizable" before disposal
  - Waste manifest required
  - Waste logged in Metrc before disposal
  - Ohio EPA coordination for certain waste streams

### 8.2 Compliance Checklist — Waste

- [ ] All waste rendered unusable before leaving restricted area (Admin. Code 3780-3-15(A))
- [ ] Waste logged in Metrc with destruction reason code (Admin. Code 3780-3-15(C))
- [ ] Waste manifests retained minimum 3 years (Admin. Code 3780-3-15(B))
- [ ] Ohio EPA coordination for any hazardous waste streams from processing (ORC § 3734)
- [ ] Third-party waste hauler authorized under Ohio law (Admin. Code 3780-3-15(D))
