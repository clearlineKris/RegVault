## Part 1: Adult-Use Transition History — June 2024 to Present

### Launch Context
Ohio's adult-use market launched June 7, 2024, under rules that were finalized in compressed timeframes. Many operators transitioned from existing medical marijuana licenses; others entered as new adult-use licensees. The transition created a dual-track compliance environment that is still being navigated in 2026.

---

### Change 1: Adult-Use Framework Activation — June 2024

| Field | Detail |
|---|---|
| **Rule** | ORC Chapter 3780; Ohio Admin. Code Chapter 3780 (full adult-use rule set) |
| **Effective Date** | June 7, 2024 |
| **Operator Impact** | All adult-use license holders obligated under new rules from day one |
| **Key New Requirements** | Metrc adult-use enrollment; packaging/labeling per Admin. Code 3780-3-10; security per Admin. Code 3780-3-06; separate Transporter license requirement |
| **Transition Gap** | Medical-era labels, systems, and SOPs not automatically compliant with adult-use rules — operators required to update immediately |
| **Action Required** | Full compliance audit against Admin. Code 3780 standards for all adult-use operations |

---

### Change 2: Transporter License Mandate Clarified — 2025

| Field | Detail |
|---|---|
| **Rule** | ORC § 3780.13 |
| **Clarification** | ODCC clarified that all inter-facility product movement — including by the originating licensee between their own facilities — requires an ODCC Transporter license |
| **Enforcement Status** | Active |
| **Operator Impact** | Multi-facility operators must hold or contract a licensed transporter for every product movement, including to testing facilities |
| **Action Required** | Verify Transporter license coverage for all product movement scenarios; identify gaps; either obtain license or contract a licensed transporter |

---

### Change 3: Dual-Track Metrc Segregation Standard — 2025

| Field | Detail |
|---|---|
| **Rule** | Ohio Admin. Code 3780-4-02(F) |
| **Clarification** | ODCC enforcement activity clarified that dual-license operators must maintain physical AND digital segregation of medical and adult-use inventory |
| **Enforcement Status** | Active — generating corrective actions |
| **Operator Impact** | Dual-track operators with shared cultivation/processing space face the highest exposure |
| **Action Required** | Physical space designation + ODCC documentation of segregation methodology |

---

### Change 4: Packaging/Labeling Compliance Focus — 2024–2025

| Field | Detail |
|---|---|
| **Rule** | Ohio Admin. Code 3780-3-10 |
| **Enforcement Signal** | ODCC early retail inspections concentrated on packaging and labeling; highest corrective action category in 2024–2025 |
| **Primary Violations** | Medical-era labels in adult-use inventory; batch/lot number format errors; per-serving THC notation inconsistencies |
| **Action Required** | Label audit; remove non-compliant products; implement label QA checkpoint |
