## Section 1: Licensing Obligations

### 1.1 Statutory Foundation
- **CRS § 44-10-301**: Retail Marijuana Establishment Licensing  
  - Pinpoint: (2)(a)–(f) — application categories, owner disclosure, financial source documentation  
  - Pinpoint: (3)(a) — annual renewal timelines (90-day pre-expiration window)  
- **CRS § 44-10-401**: Medical Marijuana Center Licensing  
  - Pinpoint: (1)(b) — physician recommendation verification at point of sale  
- **1 CCR 212-3, Rule 2-205** (eff. Jan 5, 2026): Updated license transfer procedures; new 30-day pre-sale notification to MED  
- **1 CCR 212-3, Rule 2-230**: Temporary modifications — structures, premises changes, and event licenses  

### 1.2 License Categories Operative in 2026

| License Type | Governing Rule | 2026 Update |
|---|---|---|
| Retail Marijuana Store | 1 CCR 212-3, R. 2-205 | Transfer notice window extended to 30 days |
| Retail Marijuana Cultivation Facility | 1 CCR 212-3, R. 3-215 | Canopy cap tracking tied to METRC batch creation |
| Retail MIP (Manufacturing) | 1 CCR 212-3, R. 4-120 | Solvent extraction SOPs must be MED-filed |
| Marijuana Testing Facility | 1 CCR 212-3, R. 5-110 | ISO 17025 accreditation required by July 1, 2026 |
| Medical Marijuana Business | 1 CCR 212-3, R. 2-120 | Dual-use facility overlaps — joint inspection protocols active |
| Retail Delivery Operator | 1 CCR 212-3, R. 2-260 (NEW) | Delivery manifest and geofencing requirements in effect |

### 1.3 Compliance Checklist — Licensing

- [ ] Current license certificate posted at licensed premises (Rule 2-205(C))  
- [ ] Ownership structure disclosed to MED; no undisclosed beneficial owners (CRS § 44-10-307)  
- [ ] Key license holder background checks current within 3 years (Rule 2-215)  
- [ ] Renewal application filed ≥ 90 days prior to expiration (Rule 2-230(B))  
- [ ] Municipal license maintained and current alongside state license (CRS § 44-10-104)  
- [ ] License modification filed for any structural premises change (Rule 2-230(A))

## Section 2: Seed-to-Sale Tracking

### 2.1 Statutory Foundation
- **CRS § 44-10-203(3)**: Seed-to-sale tracking system mandate  
- **1 CCR 212-3, Rule 3-105**: Tracking System Participation  
  - Pinpoint: (B)(1)–(4) — device requirements, access credentialing, reconciliation cadence  
- **Bulletin 26:01 (01/23/2026)**: MED clarification on METRC entry windows — same-business-day rule reaffirmed; MED will not accept "overnight batch" entry as compliant absent documented system outage  

### 2.2 METRC Obligations — 2026 Operative Rules

| Obligation | Citation | Deadline |
|---|---|---|
| Plant tag assignment at cloning/seed | Rule 3-105(B)(1) | Day of propagation |
| Harvest batch creation | Rule 3-105(B)(2) | Day of harvest |
| Transfer manifest creation | Rule 3-115(A) | Before vehicle departure |
| Inventory reconciliation | Rule 3-105(B)(4) | Weekly; documented |
| System outage reporting to MED | Bulletin 26:01 | Within 24 hours of outage |
| METRC API integration (3rd-party POS) | Rule 3-105(C) | Real-time push required |

### 2.3 Compliance Checklist — Tracking

- [ ] All plants tagged at first transplant or germination (Rule 3-105(B)(1))  
- [ ] Harvest batches created same-day (Bulletin 26:01)  
- [ ] Transfer manifests generated before transport — no verbal-only transfers (Rule 3-115)  
- [ ] Weekly physical count reconciled against METRC (Rule 3-105(B)(4))  
- [ ] Discrepancy reports filed within 3 business days of discovery (Rule 3-105(D))  
- [ ] METRC credentials active for all badged employees with inventory access (Rule 3-105(B)(3))

## Section 3: Security & Access Control

### 3.1 Statutory Foundation
- **1 CCR 212-3, Rule 3-225**: Premises Security  
  - Pinpoint: (A) — camera coverage minimums (all entry/exit, all storage, all point-of-sale)  
  - Pinpoint: (B) — 40-day video retention requirement  
  - Pinpoint: (C) — visitor log and escort requirements  
- **1 CCR 212-3, Rule 3-235**: Alarm Systems — panic buttons, motion detection, central monitoring  
- **2025 Enforcement Report (02/19/2026)**: Camera blind spots and visitor log gaps cited in 23% of corrective action plans issued in 2025  

### 3.2 Security Requirements Matrix — 2026

| Area | Camera Required | Lighting Required | Restricted Access |
|---|---|---|---|
| All exterior entry/exit points | ✅ | ✅ | ✅ Badged only |
| Retail sales floor | ✅ | ✅ | ✅ Employees + licensed visitors |
| Inventory/vault storage | ✅ | ✅ | ✅ Managers only per Rule 3-225(C) |
| MIP extraction rooms | ✅ | ✅ | ✅ Licensed MIP staff only |
| Cultivation canopy | ✅ | ✅ | ✅ Badged only |
| Loading/unloading docks | ✅ | ✅ | ✅ Manifest-verified personnel |
| Break rooms / HR areas | ❌ | N/A | N/A |

### 3.3 Compliance Checklist — Security

- [ ] DVR/NVR system retaining footage for minimum 40 days (Rule 3-225(B))  
- [ ] All camera feeds confirmed to cover required areas without blind spots (Rule 3-225(A))  
- [ ] Alarm system under 24/7 central monitoring contract (Rule 3-235(B))  
- [ ] Visitor log maintained with date/time/purpose/escort name (Rule 3-225(C))  
- [ ] Employee badge program — all staff badged before unsupervised access (Rule 3-305(A))  
- [ ] Annual security audit documented and on file (Rule 3-225(D))

## Section 4: Pesticide Compliance

### 4.1 Statutory Foundation
- **CRS § 44-10-203(2)(h)**: Pesticide use restrictions — cannabis-specific prohibition list  
- **1 CCR 212-3, Rule 3-1005**: Pesticide and Chemical Input Requirements (updated Jan 5, 2026)  
  - Pinpoint: (A) — EPA-registered only, cannabis-labeled or non-phytotoxic food-crop approved  
  - Pinpoint: (B) — application log required: product name, EPA reg. no., date, applicator, rate, target pest  
  - Pinpoint: (C) — 48-hour re-entry interval post-application in canopy  
- **CDPHE Guidance Memo (March 2026)**: Mycobutanil (Eagle 20EW) enforcement escalation — first offense now carries mandatory product quarantine pending remediation testing  

### 4.2 Pesticide Categories — Compliance Status

| Category | Status Under CO 2026 Rules | Risk Level |
|---|---|---|
| EPA-registered, food-crop labeled | ✅ Permitted — with log | Low |
| EPA-registered, non-cannabis specific | ✅ Permitted — with log + CDPHE check | Medium |
| Mycobutanil (Eagle 20EW) | 🚫 PROHIBITED — Enforcement escalated | Critical |
| Imidacloprid (systemic) | 🚫 PROHIBITED | Critical |
| Pyrethrins (contact, labeled) | ✅ Permitted — outdoor/canopy only | Low–Medium |
| Neem oil (cold-pressed, food-grade) | ✅ Generally permitted — verify label | Low |
| Sulfur-based fungicides | ⚠️ Conditional — temp/humidity restrictions | Medium |
| Hydrogen peroxide (food-grade) | ✅ Permitted — dilution rate must be logged | Low |
| Spinosad | ✅ Permitted — with log | Low |

### 4.3 Compliance Checklist — Pesticides

- [ ] Pesticide application log current and accessible for inspector review (Rule 3-1005(B))  
- [ ] All pesticides verified against MED/CDPHE approved-use list before purchase (Rule 3-1005(A))  
- [ ] No Eagle 20EW or imidacloprid on premises or in supply chain (Rule 3-1005(C); CDPHE Memo March 2026)  
- [ ] 48-hour re-entry intervals documented after each application (Rule 3-1005(C))  
- [ ] IPM (Integrated Pest Management) plan on file (Rule 3-1005(D))  
- [ ] Testing facility informed of pesticide applications before harvest batch submission (Rule 5-125)

## Section 5: Packaging & Labeling

### 5.1 Statutory Foundation
- **CRS § 44-10-203(4)**: Packaging and labeling consumer protection mandate  
- **1 CCR 212-3, Rule 3-1010**: Retail Marijuana Product Packaging and Labeling  
  - Pinpoint: (A)(1)–(6) — universal THC symbol, warning statements, net weight, license number  
  - Pinpoint: (B) — child-resistant packaging standards (16 CFR § 1700.15 compliance)  
  - Pinpoint: (C) — serving size indicators on edibles (5mg THC per serving; 100mg per package)  
- **1 CCR 212-3, Rule 3-1015** (updated Jan 5, 2026): QR code traceability on retail product — consumer-facing METRC batch linkage now required for all products manufactured after March 1, 2026  

### 5.2 Required Label Elements — 2026

| Element | Requirement | Citation |
|---|---|---|
| Universal THC symbol (red exclamation) | Required — min ¼" diameter | Rule 3-1010(A)(1) |
| "Contains Marijuana" statement | Required — prominently displayed | Rule 3-1010(A)(2) |
| Net weight or volume | Required in grams/oz and mL | Rule 3-1010(A)(3) |
| Total THC content (mg) | Required — post-activation for edibles | Rule 3-1010(A)(4) |
| Licensed manufacturer name + license no. | Required | Rule 3-1010(A)(5) |
| "Keep out of reach of children" | Required | Rule 3-1010(A)(6) |
| Batch/lot number (METRC-linked) | Required — March 1, 2026 forward | Rule 3-1015 (2026) |
| QR code linking to batch COA | Required — manufactured after 3/1/2026 | Rule 3-1015 (2026) |
| Child-resistant packaging confirmation | Required — 16 CFR § 1700.15 | Rule 3-1010(B) |

## Section 6: Testing Requirements

### 6.1 Statutory Foundation
- **CRS § 44-10-203(2)(f)**: Mandatory third-party laboratory testing  
- **1 CCR 212-3, Rule 5-110**: Testing Facility Licensing and Standards  
  - Pinpoint: ISO 17025 accreditation required by July 1, 2026  
- **1 CCR 212-3, Rule 5-120**: Required Test Panels  
  - Potency (cannabinoid profile), microbials, mycotoxins, residual solvents, pesticide residues, heavy metals, moisture content  
- **1 CCR 212-3, Rule 5-125**: Testing Hold Procedures — product under testing hold cannot be sold, transferred, or destroyed without MED authorization  

### 6.2 Required Testing Matrix — 2026

| Test Category | Required For | Fail Threshold | Citation |
|---|---|---|---|
| Potency (cannabinoid) | All products | Label accuracy ±15% | Rule 5-120(A) |
| Microbials (total aerobic count, e-coli, salmonella) | All products | State action limits | Rule 5-120(B) |
| Mycotoxins (aflatoxin, ochratoxin) | Flower, concentrates | 20 ppb total | Rule 5-120(C) |
| Residual solvents | Concentrates, vapes | State action limits per solvent | Rule 5-120(D) |
| Pesticide residues | All products | MED/CDPHE action limits | Rule 5-120(E) |
| Heavy metals | All products | USP <232> limits | Rule 5-120(F) |
| Moisture content | Flower | ≤ 15% (mold risk threshold) | Rule 5-120(G) |

## Section 7: Transportation

### 7.1 Statutory Foundation
- **1 CCR 212-3, Rule 3-115**: Marijuana Transport Requirements  
  - Pinpoint: (A) — manifest required before departure  
  - Pinpoint: (B) — vehicle must be enclosed, locked cargo area  
  - Pinpoint: (C) — driver must be badged MED employee; no contract drivers  
  - Pinpoint: (D) — manifests must match METRC transfer entries  
- **Bulletin 26:01 (01/23/2026)**: Manifest discrepancies between METRC and paper manifest now automatic corrective action trigger  

### 7.2 Compliance Checklist — Transport

- [ ] METRC transfer record created and approved before vehicle departs (Rule 3-115(A))  
- [ ] Vehicle inspection log on file — locked cargo compartment verified (Rule 3-115(B))  
- [ ] Driver badge active and current (Rule 3-115(C))  
- [ ] Paper manifest matches METRC manifest — no discrepancies tolerated (Bulletin 26:01)  
- [ ] Delivery operator vehicles geofenced per Rule 2-260 routing requirements (Rule 2-260)

## Section 8: Waste Handling

### 8.1 Statutory Foundation
- **1 CCR 212-3, Rule 3-1505**: Marijuana Waste Disposal  
  - Pinpoint: (A) — rendering "unusable and unrecognizable" before disposal  
  - Pinpoint: (B) — waste manifest required for transport  
  - Pinpoint: (C) — waste must be logged in METRC before disposal  
  - Pinpoint: (D) — composting permitted only at licensed disposal facilities  

### 8.2 Compliance Checklist — Waste

- [ ] All waste rendered unusable before leaving restricted area (Rule 3-1505(A))  
- [ ] Waste logged in METRC with destruction reason code (Rule 3-1505(C))  
- [ ] Waste manifests retained for minimum 3 years (Rule 3-1505(B))  
- [ ] Third-party waste hauler holding appropriate MED authorization (Rule 3-1505(D))  
- [ ] No untreated cannabis waste in general trash stream (Rule 3-1505(A))
