## Section C: Application Log — Required Fields (19 CSR 100-4.010)

| Field | Required | Example |
|---|---|---|
| Product name (full) | ✅ | "Pyganic EC 1.4" |
| EPA Registration Number | ✅ | "EPA Reg. No. 1021-1771-67690" |
| Date and time | ✅ | "April 14, 2026 – 08:30 AM" |
| **Applicator name + MDA license number** | ✅ | "J. Smith, MDA Lic. #P-12345" |
| Application rate | ✅ | "2 fl oz per gallon" |
| Application method | ✅ | "Backpack sprayer, foliar" |
| Target pest/disease | ✅ | "Thrips (Frankliniella occidentalis)" |
| Zone/area | ✅ | "Canopy Room B, rows 1–6" |
| Re-entry interval | ✅ | "48 hours — access restricted until 4/16 08:30 AM" |

## Section D: MDA Applicator License Verification

| Party | Required MDA Credential | Verification |
|---|---|---|
| In-house IPM staff | MDA Commercial Pesticide Applicator license | MDA online license lookup |
| IPM contractor | MDA Pesticide Business License + applicator | MDA + copy on file |
| Out-of-state contractors | Missouri license required | Verify with MDA |
