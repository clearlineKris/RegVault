## Michigan Regulatory Framework Overview

**Governing Authorities:**
- **Cannabis Regulatory Agency (CRA)**: Primary adult-use and medical licensing, inspection, and enforcement authority
- **MRTMA**: Michigan Regulation and Taxation of Marihuana Act — MCL 333.27951 et seq. — governs adult-use
- **MMFLA**: Michigan Medical Marihuana Facilities Licensing Act — MCL 333.27101 et seq. — governs medical
- **CRA Administrative Rules (MARC)**: R 420.1 et seq. — operational compliance requirements

Michigan operates a dual-track system: adult-use (MRTMA) and medical (MMFLA) licenses, both under CRA jurisdiction. Many operators hold licenses on both tracks.

## Section 1: Licensing Obligations

### 1.1 License Categories — Adult-Use (MRTMA) 2026

| License Type | Rule Ref | Key Limit / Condition |
|---|---|---|
| Class A Grower | R 420.204 | ≤ 100 plants |
| Class B Grower | R 420.204 | ≤ 500 plants |
| Class C Grower | R 420.204 | ≤ 2,000 plants; up to 3 of same type |
| Processor | R 420.206 | Conversion of flower/trim to products |
| Retailer | R 420.207 | Adult-use retail sales |
| Microbusiness | R 420.210 | Grow (≤150 plants) + process + retail; self-supply only |
| Safety Compliance Facility | R 420.213 | Testing laboratory |
| Secure Transporter | R 420.212 | Inter-facility product movement |
| Designated Consumption Establishment | R 420.217 | On-premises consumption |
| Class C Stacking | MCL 333.27959(3) | Up to 3 Class C licenses per licensee — separate operations required |

### 1.2 Municipal Opt-In Requirement — MCL 333.27956
Michigan cannabis licenses require local authorization. Municipalities may opt-in (allow) or opt-out (prohibit). A state license does not authorize operation in a municipality that has not opted in. Opt-in status can change — municipalities can modify or revoke local authorization.

### 1.3 Compliance Checklist — Licensing

- [ ] CRA license certificate posted at licensed premises (R 420.3(1))
- [ ] Municipal opt-in authorization current and accessible (MCL 333.27956)
- [ ] All principal officers, board members, and key employees disclosed to CRA (R 420.9)
- [ ] Background checks current for all required personnel (R 420.10)
- [ ] Annual renewal filed within CRA renewal window
- [ ] Class C stacking operators: separate operations documentation current (MCL 333.27959(3))

## Section 2: Seed-to-Sale Tracking (Metrc)

### 2.1 Statutory Foundation
- **MCL 333.27962**: Mandatory statewide monitoring system
- **CRA Administrative Rules R 420.301 et seq.**: Metrc tracking requirements
- Michigan uses **Metrc** as its statewide cannabis tracking system

### 2.2 Metrc Obligations — 2026

| Obligation | Rule Ref | Timeline |
|---|---|---|
| Plant tag at propagation | R 420.302(1) | Day of propagation |
| Harvest batch creation | R 420.302(3) | Day of harvest |
| Transfer manifest — pre-departure | R 420.305 | Before vehicle moves |
| Inventory reconciliation | R 420.302(7) | Weekly; documented |
| Dual-track reconciliation (medical/adult-use) | R 420.302(9) | Daily for dual-track operators |
| Discrepancy report to CRA | R 420.302(8) | Within 24 hours of discovery |

**Michigan-specific:** Michigan's discrepancy reporting window is **24 hours** — more stringent than Ohio's 3-business-day window. This is a field-level compliance pressure point.

### 2.3 Compliance Checklist — Tracking

- [ ] All plants tagged at propagation (R 420.302(1))
- [ ] Harvest batches created same-day (R 420.302(3))
- [ ] Transfer manifests generated pre-departure (R 420.305)
- [ ] Weekly physical inventory count reconciled against Metrc (R 420.302(7))
- [ ] Discrepancy reports filed within **24 hours** of discovery (R 420.302(8))
- [ ] Dual-track (medical/adult-use) Metrc accounts segregated (R 420.302(9))

## Section 3: Security & Access Control

### 3.1 Rule Foundation
- **CRA Administrative Rules R 420.502**: Security requirements
  - Camera coverage: all entry/exit, all storage, all canopy, all sales areas
  - Retention: **90-day** minimum
  - Visitor log and restricted access requirements
- **R 420.503**: Alarm systems — 24/7 monitoring, panic buttons, motion detection

### 3.2 Security Requirements Matrix

| Area | Camera Required | Retention | Restricted |
|---|---|---|---|
| All exterior entry/exit | ✅ | 90 days | ✅ |
| Retail sales floor | ✅ | 90 days | ✅ |
| Vault/storage | ✅ | 90 days | ✅ |
| Processing rooms | ✅ | 90 days | ✅ |
| Cultivation canopy | ✅ | 90 days | ✅ |
| Loading/unloading | ✅ | 90 days | ✅ |

### 3.3 Compliance Checklist — Security

- [ ] 90-day footage retention — storage capacity verified at current camera count and resolution (R 420.502(2))
- [ ] Camera coverage audit against current facility footprint
- [ ] Visitor log with name, date, time, purpose, escort employee
- [ ] 24/7 alarm monitoring contract current (R 420.503)
- [ ] Security plan updated within 14 days of any material premises change (R 420.502(5))

## Section 4: Pesticide Compliance

### 4.1 Rule Foundation
- **CRA Administrative Rules R 420.402**: Pesticide requirements
  - Only EPA-registered pesticides permitted; labeled for cannabis, food crops, or non-phytotoxic
  - Application log: product, EPA reg. no., date, applicator, rate, target pest, area
  - Michigan Department of Agriculture and Rural Development (MDARD) pesticide applicator license required for commercial applications
  - 48-hour re-entry interval in canopy after application
  - IPM plan required; on file with CRA

### 4.2 Compliance Checklist — Pesticides

- [ ] All pesticides EPA-registered and verified as permitted (R 420.402(1))
- [ ] Application log current, specific (R 420.402(3))
- [ ] MDARD pesticide applicator license verified for all commercial applicators
- [ ] 48-hour re-entry intervals documented (R 420.402(4))
- [ ] IPM plan on file with CRA, current version (R 420.402(6))

## Section 5: Packaging & Labeling

### 5.1 Rule Foundation
- **CRA Administrative Rules R 420.601 et seq.**: Product packaging standards
  - Child-resistant packaging (16 CFR § 1700.15)
  - Michigan universal THC symbol required
  - Net weight; THC content per serving and per package
  - Licensed manufacturer + CRA license number
  - Batch/lot number linked to Metrc

### 5.2 Required Label Elements

| Element | Requirement |
|---|---|
| Michigan universal THC symbol | ✅ Required |
| "Contains Marijuana" / "Adults 21+" | ✅ Required |
| Net weight or volume | ✅ Required |
| Total THC per serving (mg) — edibles | ✅ Required |
| Total THC per package (mg) | ✅ Required |
| Licensed manufacturer + CRA license no. | ✅ Required |
| Batch/lot number (Metrc-linked) | ✅ Required |
| Child-resistant packaging | ✅ Required |

| Heavy metals | All products |
| Water activity / moisture | Flower |
| Terpene profile (optional) | Voluntary |
