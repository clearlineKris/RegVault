## Section C: Application Log — Required Fields (R 420.402(3))

| Field | Required | Example |
|---|---|---|
| Product name (full) | ✅ | "Pyganic EC 1.4" |
| EPA Registration Number | ✅ | "EPA Reg. No. 1021-1771-67690" |
| Date and time | ✅ | "April 14, 2026 – 08:30 AM" |
| **Applicator name + MDARD license number** | ✅ **Michigan-specific** | "J. Smith, MDARD Lic. #A12345" |
| Application rate | ✅ | "2 fl oz per gallon" |
| Application method | ✅ | "Backpack sprayer, foliar" |
| Target pest/disease | ✅ | "Thrips (Frankliniella occidentalis)" |
| Zone/area | ✅ | "Canopy Room B, rows 1–6" |
| Re-entry interval | ✅ | "48 hours — access restricted until 4/16 08:30 AM" |

## Section D: MDARD Applicator License Verification

| Party | Required MDARD Credential | Verification Method |
|---|---|---|
| In-house IPM staff | MDARD Commercial Pesticide Applicator license | MDARD online lookup |
| IPM contractor | MDARD Pesticide Business License + applicator | MDARD online + copy on file |
| Out-of-state contractors | Must hold Michigan reciprocity / Michigan license | Confirm with MDARD |
