## Pennsylvania Pesticide Compliance Framework

**Two-authority structure:**
1. **OMM/DOH (28 Pa. Code § 1141.38)**: Medical marijuana pesticide rules — permitted products, application log, re-entry intervals, IPM plan
2. **PDA (Pennsylvania Pesticide Control Act, 3 P.S. § 111.21 et seq.)**: Commercial pesticide applicator certification — applies to cannabis Grower/Processors as agricultural operations

---

## PDA Applicator Certification Requirement

Pennsylvania's Pesticide Control Act requires that commercial pesticide applications at a business operation be made by or under the direct supervision of a PDA-certified commercial pesticide applicator. Cannabis Grower/Processors are covered.

**Verification requirements:**
- In-house cultivation staff who apply pesticides must hold PDA Commercial Pesticide Applicator certification
- IPM contractors must hold PDA Pesticide Business License + appropriate applicator certification
- Annual renewal; verify current status before every application

## Application Log — Required Fields (28 Pa. Code § 1141.38)

| Field | Required |
|---|---|
| Product name + EPA Registration No. | ✅ |
| Date and time | ✅ |
| Applicator name + **PDA certification number** | ✅ **Pennsylvania-specific** |
| Application rate and method | ✅ |
| Target pest/disease | ✅ |
| Zone/area (note: reference BioTrack lot if applicable) | ✅ |
| Re-entry interval (48 hours) | ✅ |

**Pennsylvania-specific log note:** The application log should reference the BioTrack lot number for the affected batch when applicable — this creates a traceable link between pesticide application records and the tracking record for OMM inspection purposes.

## Compliance Hardening Checklist — Pennsylvania

- [ ] All pesticides on premises EPA-registered and on OMM/PDA permitted list (28 Pa. Code § 1141.38)
- [ ] Application log current, complete, with PDA certification numbers
- [ ] All commercial applicators hold current PDA certification
- [ ] IPM contractors verified for PDA Pesticide Business License + applicator certification
- [ ] 48-hour re-entry intervals documented and enforced
- [ ] IPM plan on file with OMM, current version
- [ ] Application log entries reference BioTrack lot numbers where applicable
- [ ] Supplier pesticide disclosure obtained for all plant/clone purchases
