## Pennsylvania Regulatory Framework Overview

**Governing Authority:** Pennsylvania Department of Health (DOH), Office of Medical Marijuana (OMM).

Pennsylvania operates a **medical-only** program under the Medical Marijuana Act (MMA), Act 16 of 2016. As of April 2026, Pennsylvania has not enacted adult-use legalization. The MMA framework governs all licensed cannabis activity.

**Critical Pennsylvania distinction:** Pennsylvania uses **BioTrack THC** — not Metrc — as its seed-to-sale tracking platform. This is a fundamental operational difference from most other active cannabis states. Operators moving into Pennsylvania from Metrc states must retrain staff and rebuild tracking SOPs.

## Section 1: Licensing Obligations

### 1.1 Statutory Foundation
- **35 P.S. § 10231.301**: Grower/Processor license
- **35 P.S. § 10231.401**: Dispensary organization license
- **35 P.S. § 10231.501**: Clinical Registrant license (research partnership with academic medical center)
- **28 Pa. Code § 1141.21 et seq.**: Grower/Processor operational requirements
- **28 Pa. Code § 1141.51 et seq.**: Dispensary operational requirements

### 1.2 License Categories — Pennsylvania 2026

| License Type | Statute | Key Condition |
|---|---|---|
| Grower/Processor | 35 P.S. § 10231.301 | Combined cultivation + processing under one license |
| Dispensary Organization | 35 P.S. § 10231.401 | Up to 3 dispensary locations per organization |
| Dispensary Organization — Expanded | DOH approval | Up to 5 locations with OMM approval |
| Clinical Registrant | 35 P.S. § 10231.501 | Must partner with an academic medical center |
| Delivery | DOH regulations | Home delivery to registered patients |

**Pennsylvania-specific structure:** Unlike most states, Pennsylvania issues Dispensary *Organization* licenses — each organization may operate up to 3 dispensary locations (or up to 5 with OMM approval). This is a single license covering multiple locations.

### 1.3 Patient Registry and Verification
- Patients must be certified by a DOH-approved physician (35 P.S. § 10231.303)
- Certified patients receive a Medical Marijuana ID Card
- Dispensaries must verify ID Card at point of sale — every transaction
- Caregivers (registered caregivers) may purchase on behalf of patients
- **Out-of-state patients:** Pennsylvania has limited reciprocity provisions — verify current OMM guidance

### 1.4 Compliance Checklist — Licensing

- [ ] Current OMM license certificate posted at each licensed location (28 Pa. Code § 1141.53(a))
- [ ] All principal owners, board members, and key employees disclosed to OMM (35 P.S. § 10231.301(e))
- [ ] Background checks current for all required personnel
- [ ] Annual permit renewal filed within OMM renewal window
- [ ] Dispensary organization: all operating locations within licensed limit (3 or 5)
- [ ] Patient ID Card verification process operational at every dispensary location

## Section 2: Seed-to-Sale Tracking (BioTrack THC)

### 2.1 Statutory Foundation
- **35 P.S. § 10231.701**: Mandatory tracking system participation
- **28 Pa. Code § 1141.35**: BioTrack THC tracking requirements for Grower/Processors
- **28 Pa. Code § 1141.57**: BioTrack tracking requirements for Dispensaries

**⚠️ Pennsylvania uses BioTrack THC — NOT Metrc.** This is the most important operational distinction for operators entering Pennsylvania from other markets.

### 2.2 BioTrack Obligations — 2026

| Obligation | Rule Ref | Timeline |
|---|---|---|
| Plant tag at propagation | 28 Pa. Code § 1141.35(a) | Day of propagation |
| Harvest lot creation | 28 Pa. Code § 1141.35(b) | Day of harvest |
| Transfer manifest | 28 Pa. Code § 1141.35(d) | Before vehicle moves |
| Inventory reconciliation | 28 Pa. Code § 1141.35(e) | Weekly; documented |
| Dispensary receipt posting in BioTrack | 28 Pa. Code § 1141.57(b) | At time of receipt |
| Patient sale recorded in BioTrack | 28 Pa. Code § 1141.57(c) | At time of sale |
| Discrepancy report | 28 Pa. Code § 1141.35(f) | Within 3 business days |

### 2.3 Pennsylvania-Specific: BioTrack vs. Metrc Operational Differences

| Function | BioTrack (PA) | Metrc (most states) |
|---|---|---|
| Tracking granularity | Lot-based | Package/tag based |
| Transfer process | Manifest in BioTrack | Metrc transfer with RFID tags |
| Patient sale recording | BioTrack integrated or manual | Via Metrc API/POS |
| Inventory structure | Lot hierarchy | Individual package hierarchy |

### 2.4 Compliance Checklist — Tracking

- [ ] All plants entered in BioTrack at propagation (28 Pa. Code § 1141.35(a))
- [ ] Harvest lots created same-day (28 Pa. Code § 1141.35(b))
- [ ] Transfer manifests generated pre-departure (28 Pa. Code § 1141.35(d))
- [ ] Weekly inventory reconciliation against BioTrack (28 Pa. Code § 1141.35(e))
- [ ] Every patient sale recorded in BioTrack at time of transaction (28 Pa. Code § 1141.57(c))
- [ ] Discrepancy reports filed within 3 business days (28 Pa. Code § 1141.35(f))

## Section 3: Security & Access Control

### 3.1 Rule Foundation
- **28 Pa. Code § 1141.25 (G/P) and § 1141.53(i) (Dispensary)**: Security requirements
  - Camera coverage: all entry/exit, all storage, all sales, all cultivation, all processing areas
  - Retention: **90-day** minimum
  - Visitor log: all non-employee visitors
  - Alarm systems: 24/7 central monitoring

### 3.2 Security Requirements Matrix

| Area | Camera Required | Retention | Restricted Access |
|---|---|---|---|
| All exterior entry/exit | ✅ | 90 days | ✅ |
| Dispensary sales floor | ✅ | 90 days | ✅ |
| Vault/storage | ✅ | 90 days | ✅ Manager access |
| Processing/extraction | ✅ | 90 days | ✅ |
| Cultivation canopy | ✅ | 90 days | ✅ |
| Loading/receiving | ✅ | 90 days | ✅ |

## Section 4: Pesticide Compliance

### 4.1 Rule Foundation
- **28 Pa. Code § 1141.38**: Chemical input and pesticide requirements for Grower/Processors
  - EPA-registered pesticides only
  - Pennsylvania Department of Agriculture (PDA) pesticide applicator certification required for commercial applications
  - Application log required
  - 48-hour re-entry interval
  - IPM plan on file with OMM

---

## Section 5: Packaging & Labeling

### 5.1 Rule Foundation
- **28 Pa. Code § 1141.45**: Medical marijuana product labeling
  - Child-resistant packaging
  - Pennsylvania OMM required labeling elements
  - THC and CBD content per dose and per package
  - "For Medical Use Only" statement (medical program requirement)
  - Grower/Processor name + OMM permit number
  - BioTrack lot number

**Pennsylvania-specific:** Because Pennsylvania is medical-only, all labels must include "For Medical Use Only" and reference the patient-directed dosing framework. Adult-use-style labels are not used.

## Section 6: Testing Requirements

| Test Category | Required For |
|---|---|
| Potency (THC/CBD profile) | All products |
| Microbials | All products |
| Mycotoxins | Flower, concentrates |
| Residual solvents | Concentrates, vapes |
| Pesticide residues | All products |
| Heavy metals | All products |
| Water activity / moisture | Flower |

---

## Section 7: Forms and Dispensary Requirements

- **Patient ID Card verification** at every sale (35 P.S. § 10231.405)
- **Daily sales limit per patient**: 30-day supply as recommended by certifying practitioner
- **Patient consultation**: Dispensaries must have a pharmacist on site or available for consultation (28 Pa. Code § 1141.55)
- **Pharmacist requirement**: This is a Pennsylvania-specific requirement not present in most states — every dispensary organization must have a pharmacist involved in operations
